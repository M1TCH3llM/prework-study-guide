HTML

  • The head element contains information about the webpage.
  • he body element represents the visible content shown to the user.
  • Declorations don't have to be case sensitive.
  • Alt text used for screan readers = alt text "".
  • Basic codes
    • head = meta data for web browser.
    • body = visible for user.
    • lang = language... en = English
    • Img = Image... src = image source
    • h1,h2-h6 = heading sizes
    • br = page break
    • a = anchor, used for links

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Inline CSS: Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
    • Code: “Style=”
  • Internal style sheet: contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
    • Code: “< style >h1{color:red;}"
  • External CSS style sheet: allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • rel: attribute specifies the relationship between the current document and the linked document resource.
  • href: attribute specifies the location (URL) of the external resource.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main code
  • git pull origin main: pulls main data from repo
  • Steps to start branch
    • git checkout main; goes to main branch
    • git pull origin main; syncs main branch
    • Navigate to desiered folder; cd 'folder name'
    • Or create new folder; touch 'folder name'
    • git checkout -b 'new branch name'; creates a new branch
  • Steps to push.
    • git add -A; Stages Changes
    • git commit -m "comment"; add description
    • git pull origin main; checks if main branch is up to date,
    • git push origin .....; pushes branch to GitHub
